  - title: "Einen String einlesen"
    type: "Programming"
    order: 3
    description: "In Rust nutzt man `std::io::stdin()`, um Eingaben einzulesen. <br>Lies einen Namen von der Tastatur ein und gib 'Hallo [Name]!' aus. <br>Hinweis: Tippe in das Eingabefeld rechts das Wort 'Rust' ein, bevor du auf Ausführen klickst."
    initialCode: |
      use std::io;

      fn main() {
          let mut input = String::new(); // Erstellt einen Puffer
          
          // Deine Aufgabe: Nutze io::stdin().read_line(&mut input).unwrap();
          
          println!("Hallo {}!", input.trim());
      }
    expectedOutput: "Hallo Rust!"
    solution: |
      use std::io;

      fn main() {
          let mut input = String::new();
          io::stdin().read_line(&mut input).unwrap();
          println!("Hallo {}!", input.trim());
      }
    requiredKeywords: "read_line"
  - title: "Zahl einlesen"
    type: "Programming"
    order: 3
    description: "Lies eine Zahl von der Tastatur ein, verdopple sie und gib das Ergebnis aus. <br>Hinweis: Nutze `.trim().parse::<i32>().unwrap()`, um den String in eine Zahl zu wandeln. <br>Tippe die Zahl '21' in das Eingabefeld."
    initialCode: |
      use std::io;

      fn main() {
          let mut input = String::new();
          io::stdin().read_line(&mut input).unwrap();
          
          // Deine Aufgabe: Wandle 'input' in eine Zahl 'n' um
          // let n: i32 = ...
          
          let ergebnis = n * 2;
          println!("Ergebnis: {}", ergebnis);
      }
    expectedOutput: "Ergebnis: 42"
    solution: |
      use std::io;

      fn main() {
          let mut input = String::new();
          io::stdin().read_line(&mut input).unwrap();
          let n: i32 = input.trim().parse().unwrap();
          let ergebnis = n * 2;
          println!("Ergebnis: {}", ergebnis);
      }
    requiredKeywords: "parse"

      use std::io;

      fn main() {
          let mut input = String::new();
          io::stdin().read_line(&mut input).unwrap();
          let mut parts = input.split_whitespace();
          let a: i32 = parts.next().unwrap().parse().unwrap();
          let b: i32 = parts.next().unwrap().parse().unwrap();
          println!("Summe: {}", a + b);
      }
    requiredKeywords: "split_whitespace"
---
Rust legt großen Wert auf Sicherheit. Daher ist die Standardeingabe etwas ausführlicher als in anderen Sprachen. Das Modul `std::io` bietet Zugriff auf die Funktionen, um von der Tastatur zu lesen.
